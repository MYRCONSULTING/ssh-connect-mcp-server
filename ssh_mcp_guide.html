SSH MCP Server Guide

This guide provides examples of how to use the SSH MCP server with Cline.

Note: The SSH MCP server must be properly configured in your MCP settings file with the appropriate SSH connection parameters.

Available Tools

connect

Connect to an SSH server using the configured credentials.

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="connect", 
    arguments={}
)

You can also override the environment variables by providing arguments:

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="connect", 
    arguments={
        "host": "example.com",
        "port": 22,
        "username": "user",
        "password": "pass"
    }
)

execute

Execute a command on the SSH server.

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="execute", 
    arguments={
        "command": "ls -la",
        "timeout": 60  # optional, default is 60 seconds
    }
)

upload

Upload a file to the SSH server.

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="upload", 
    arguments={
        "local_path": "/path/to/local/file",
        "remote_path": "/path/to/remote/file"
    }
)

download

Download a file from the SSH server.

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="download", 
    arguments={
        "remote_path": "/path/to/remote/file",
        "local_path": "/path/to/local/file"
    }
)

list_files

List files in a directory on the SSH server.

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="list_files", 
    arguments={
        "path": "/path/to/directory"
    }
)

disconnect

Disconnect from the SSH server.

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="disconnect", 
    arguments={}
)

Complete Workflow Example

Here's a complete example workflow:

  1. Connect to the SSH server:

    use_mcp_tool(server_name="ssh-connect", tool_name="connect", arguments={})

  2. Execute a command:

    use_mcp_tool(server_name="ssh-connect", tool_name="execute", arguments={"command": "ls -la"})

  3. Upload a file:

    use_mcp_tool(server_name="ssh-connect", tool_name="upload", arguments={"local_path": "example.txt", "remote_path": "/home/user/example.txt"})

  4. List files to verify upload:

    use_mcp_tool(server_name="ssh-connect", tool_name="list_files", arguments={"path": "/home/user"})

  5. Download a file:

    use_mcp_tool(server_name="ssh-connect", tool_name="download", arguments={"remote_path": "/home/user/example.txt", "local_path": "downloaded_example.txt"})

  6. Disconnect from the SSH server:

    use_mcp_tool(server_name="ssh-connect", tool_name="disconnect", arguments={})

Important: Always disconnect from the SSH server when you're done to free up resources.
