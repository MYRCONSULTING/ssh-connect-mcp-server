SSH MCPサーバーガイド

このガイドでは、ClineでSSH MCPサーバーを使用する方法の例を提供します。

注意： SSH MCPサーバーは、適切なSSH接続パラメータを使用してMCP設定ファイルで正しく構成する必要があります。

利用可能なツール

connect

設定された認証情報を使用してSSHサーバーに接続します。

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="connect", 
    arguments={}
)

引数を提供することで環境変数を上書きすることもできます：

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="connect", 
    arguments={
        "host": "example.com",
        "port": 22,
        "username": "user",
        "password": "pass"
    }
)

execute

SSHサーバーでコマンドを実行します。

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="execute", 
    arguments={
        "command": "ls -la",
        "timeout": 60  # オプション、デフォルトは60秒
    }
)

upload

ファイルをSSHサーバーにアップロードします。

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="upload", 
    arguments={
        "local_path": "/path/to/local/file",
        "remote_path": "/path/to/remote/file"
    }
)

download

SSHサーバーからファイルをダウンロードします。

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="download", 
    arguments={
        "remote_path": "/path/to/remote/file",
        "local_path": "/path/to/local/file"
    }
)

list_files

SSHサーバー上のディレクトリ内のファイルを一覧表示します。

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="list_files", 
    arguments={
        "path": "/path/to/directory"
    }
)

disconnect

SSHサーバーから切断します。

use_mcp_tool(
    server_name="ssh-connect", 
    tool_name="disconnect", 
    arguments={}
)

完全なワークフロー例

以下は完全なワークフロー例です：

  1. SSHサーバーに接続：

    use_mcp_tool(server_name="ssh-connect", tool_name="connect", arguments={})

  2. コマンドを実行：

    use_mcp_tool(server_name="ssh-connect", tool_name="execute", arguments={"command": "ls -la"})

  3. ファイルをアップロード：

    use_mcp_tool(server_name="ssh-connect", tool_name="upload", arguments={"local_path": "example.txt", "remote_path": "/home/user/example.txt"})

  4. アップロードを確認するためにファイルを一覧表示：

    use_mcp_tool(server_name="ssh-connect", tool_name="list_files", arguments={"path": "/home/user"})

  5. ファイルをダウンロード：

    use_mcp_tool(server_name="ssh-connect", tool_name="download", arguments={"remote_path": "/home/user/example.txt", "local_path": "downloaded_example.txt"})

  6. SSHサーバーから切断：

    use_mcp_tool(server_name="ssh-connect", tool_name="disconnect", arguments={})

重要： 作業が終わったら、リソースを解放するために必ずSSHサーバーから切断してください。
